## Goal

Understand the smallest useful LLM application before building anything bigger.

## Concepts To Learn

- input
- prompt
- model response
- what your program does around the model

## Tools Or Libraries

- Python 3
- standard library only

## Tasks

1. Define the simplest LLM app flow.
2. Define what the user provides.
3. Define what the program sends to the model.
4. Define what the model returns.
5. Write down open questions.

## Expected Outputs

- a clear beginner-friendly app definition
- a reusable representation of the app flow
- a short list of open questions

## Quick Exercises

1. Write the app flow in one line using your own words.
2. Change the app idea from question answering to summarization and note what changes.
3. List three things the program does that the model itself does not do.

## Notes

- Keep this chapter simple.
- No computer vision knowledge is assumed.
- Do not build a large system yet.
